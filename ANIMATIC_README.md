## Overview

This is a **storyboard animatic video** for the 2-minute cinematic short film "Frankenstein: First Breath". The animatic provides a visual preview of all 18 shots in the production treatment.

## Video Details

- **Filename:** `Frankenstein_FirstBreath_Animatic.mp4`
- **Runtime:** 2:03 (123 seconds)
- **Resolution:** 1920x804 (2.39:1 anamorphic aspect ratio)
- **Frame Rate:** 24 fps
- **Codec:** H.264
- **File Size:** 1.1 MB

## Content

The animatic includes:

1. **Opening Title Card** (3 seconds)
   - Film title with production specs
   
2. **18 Shot Storyboards** (0:00 - 2:00)

   - Production information

## Visual Style

The animatic uses a **gothic horror aesthetic** matching the treatment specifications:

- **Color Palette:**
  - Dark teal-black backgrounds (RGB: 15, 25, 30)
  - Orange highlights for titles (RGB: 255, 200, 100)
  - Teal accents (RGB: 0, 150, 180)
  - Light blue-gray text (RGB: 200, 220, 230)

- **Typography:**
  - Clear, readable fonts
  - Shot numbers and timecodes prominently displayed
  - Technical specifications for each shot

- **Layout:**
  - Top bar: Shot number and timecode
  - Center: Scene description
  - Bottom bar: Camera and lens specifications
  - Progress bar showing position in film

The script (`create_animatic.py`) is included in the repository and can be re-run to regenerate the video with any changes to the shot list.
